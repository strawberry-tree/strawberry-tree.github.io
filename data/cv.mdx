## 자기소개서
### 버즈빌과 프론트엔드 직무를 선택한 이유

성능 최적화와 시스템 안정화가 좋은 사용자 경험의 핵심이라고 믿습니다. 프론트엔드는 사용자가 직접 체감하는 영역이기에 기술적 개선이 즉각적인 만족도로 연결됩니다. 버즈빌의 게임 광고는 로딩 지연이 이탈률로 직결되는 환경이라는 점에서, 성능 개선이 곧 비즈니스 가치인 구조가 매력적이었습니다. '어따놀래'에서 로딩 시간을 58% 단축한 경험을 게임 광고 최적화에 기여하고 싶습니다.

### 팀이 인지하지 못한 문제 발견과 해결

팀이 로딩 지연을 서버 문제로만 보던 시점에 실제 병목이 파일 크기와 포맷에 있음을 분석했습니다. 이미지 압축 라이브러리로 텍스처를 최적화해 용량을 92% 감소시켰고, 백엔드 동료의 캐싱 제안과 결합해 로딩을 6초에서 2.5초로 개선했습니다. 실시간 협업에서는 충돌이 서비스 신뢰도에 치명적이라 판단하고 인메모리 데이터베이스 기반 락 시스템으로 안정성을 확보했습니다.

### 협업 주도 경험

리더로서 디자이너가 제기한 수동 도면 입력 문제를 컴퓨터 비전 기반 자동 인식으로 해결했고, 백엔드와는 분산 락 구조를 논의하며 시스템 안정성을 확보했습니다. 기술 결정이 팀 목표와 사용자 경험에 미치는 영향을 체감하며 협업 방법을 배웠습니다.
