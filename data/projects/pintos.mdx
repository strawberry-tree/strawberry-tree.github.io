## 프로젝트 개요

Stanford 대학의 교육용 운영체제 Pintos를 기반으로 쓰레드 관리, 시스템 콜, 가상 메모리 시스템을 직접 구현하는 프로젝트입니다. 운영체제의 핵심 개념을 실제로 구현하며 시스템 프로그래밍 역량을 강화했습니다.

## 구현 내용

가상 메모리 시스템에서는 페이지 테이블 관리와 요구 페이징(Demand Paging)을 구현하여 물리 메모리 부족 문제를 해결하는 페이지 폴트 핸들러를 구현했습니다. 또한 스왑 영역 관리, 페이지 교체 알고리즘 구현, 디스크 I/O 최적화를 포함한 디스크 스왑 메커니즘을 구축했습니다.

프로세스 동기화 부분에서는 세마포어 기반 부모-자식 프로세스 동기화를 통해 fork/wait 시스템 콜을 구현하여 프로세스 생성의 안정성을 확보하고 좀비 프로세스를 방지했습니다.

이 프로젝트를 통해 운영체제 핵심 개념의 실전 구현 경험을 쌓았으며, 로우레벨 C 프로그래밍 능력을 향상시켰습니다. 메모리 관리 및 동기화 메커니즘에 대한 이해를 깊이 있게 다졌으며, 커널 레벨 디버깅 기술을 습득했습니다.
